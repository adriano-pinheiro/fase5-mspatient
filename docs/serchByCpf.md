### Buscar o paciente pelo CPF

Retorna um paciente de acordo com o CPF informado

- **Método:** `GET`
- **Ponto de acesso:** `api/v1/pacient`
- **Código HTTP:** `200 OK`

#### Parâmetro

```shell
curl --location 'http://localhost:8094/api/v1/patient/cpf?cpf=04782771053' \
--data ''
```
