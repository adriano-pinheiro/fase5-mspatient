### Listar todos os pacientes

Retorna uma lista com todos os pacientes cadastrados no sistema

- **Método:** `GET`
- **Ponto de acesso:** `api/v1/pacient`
- **Código HTTP:** `200 OK`

#### Parâmetro

```shell
curl --location 'http://localhost:8094/api/v1/patient' \
--data ''
```
