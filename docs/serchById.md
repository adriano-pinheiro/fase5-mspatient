### Buscar o paciente pelo ID

Retorna um paciente de acordo com o ID informado

- **Método:** `GET`
- **Ponto de acesso:** `api/v1/pacient`
- **Código HTTP:** `200 OK`

#### Parâmetro

```shell
curl --location 'http://localhost:8094/api/v1/patient/67b9247fb9f06e17605df7a9' \
--data ''
```
    GET  api/v1/patient/{Id}
